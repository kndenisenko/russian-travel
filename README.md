# Проектная работа #3: Путешествие по России

**Что это и для чего?**

Это третья проектная работа, выполненная в рамках Яндекс.Практикума, на курсе веб-разработки. Работа - свёрстанная динамическая страница, которая корректно отображается в полулярных разрешениях: 1280p, 1024p 768p и 320p. Такой набор разрешений нужен для того, чтобы страница корректно отображалась на обычных мониторах, экранах ноутбуков, планшетах смартфонах и мобильных телефонах. Разумеется, если разрешение экранов последних по ширине составляет 320 пикселей и выше. Страница свёрстана на основе макета из Фигмы.
